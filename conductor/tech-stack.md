# Tech Stack

## 前端開發
- **框架：** [Next.js 15+](https://nextjs.org/) (App Router) - 利用其靜態網站生成 (SSG) 功能。
- **語言：** TypeScript - 提供強型別支援。
- **測試框架：** [Vitest](https://vitest.dev/) & [React Testing Library](https://testing-library.com/docs/react-testing-library/intro/) - 採用 TDD 開發模式。

## 樣式與 UI
- **樣式工具：** [Tailwind CSS v4](https://tailwindcss.com/) - 利用其新一代引擎進行高效開發。
- **設計風格：** Premium Glassmorphism (玻璃擬態) - 結合 HSL 顏色系統與現代化漸層 (Gradient)。
- **動畫庫：** Tailwind CSS Animation Utilities (配合標準 CSS transitions)。

## 資料獲取與整合
- **API 客戶端：** [Octokit](https://github.com/octokit/octokit.js) - GitHub 官方 SDK。
- **資料處理：** 自動化 API 抓取並轉換為內部的 `Project` 型別。

## 基礎建設與部署
- **託管平台：** GitHub Pages
- **部署流程：** GitHub Actions
